# လုံခြုံရေး မူဝါဒ (Security Policy)

## တင်ပြခြင်း (Reporting)

BigTech+ system သို့မဟုတ် repository တစ်ခုတွင် လုံခြုံရေးဆိုင်ရာ ပြဿနာ တွေ့ရှိပါက private မှသာ တင်ပြပါ။ ပုံမှန် issue မဖွင့်ပါနှင့်၊ အသေးစိတ်အချက်အလက်များကို public channel များတွင် မဆွေးနွေးပါနှင့်။

**တင်ပြရန် နည်းလမ်းများ:**

1. [security@bigtechplus.io](mailto:security@bigtechplus.io) သို့ email ပို့ပါ။
2. သို့မဟုတ် repository တွင် GitHub private vulnerability reporting ဖွင့်ထားပါက ၎င်းကို အသုံးပြုပါ။

သက်ဆိုင်သော repository သို့မဟုတ် system နှင့်တကွ သိရှိပါက version သို့မဟုတ် commit၊ ပြဿနာ ပြန်လည်မဖြစ်ပေါ်စေရန် လုပ်ဆောင်ရမည့် အဆင့်များ (steps to reproduce) နှင့် သက်ရောက်မှု (impact) တို့ကို ထည့်သွင်းဖော်ပြပေးပါ။

တင်ပြချက်များကို ၄၈ နာရီအတွင်း အသိအမှတ်ပြု အကြောင်းပြန်ပြီး နောက်ထပ်ဆောင်ရွက်မည့် အဆင့်များကို ဆက်လက်အကြောင်းကြားပါမည်။
