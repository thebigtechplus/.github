# အကူအညီ (Support)

## တောင်းဆိုရမည့် နေရာ

| အကြောင်းအရာ | နေရာ |
| --- | --- |
| Bug နှင့် ချို့ယွင်းချက်များ | သက်ဆိုင်ရာ repository တွင် **Bug Report** issue ဖွင့်ပါ |
| Feature အသစ် / မြှင့်တင်မှုများ | သက်ဆိုင်ရာ repository တွင် **Feature Request** issue ဖွင့်ပါ |
| လုံခြုံရေးဆိုင်ရာ ပြဿနာများ | [SECURITY.my.md](SECURITY.my.md) အတိုင်း လုပ်ဆောင်ပါ — ပုံမှန် issue မဖွင့်ပါနှင့် |
| ကျင့်ဝတ်ဆိုင်ရာ စိုးရိမ်မှုများ | [conduct@bigtechplus.io](mailto:conduct@bigtechplus.io) |
| ကုမ္ပဏီ / product website | [www.bigtechplus.io](https://www.bigtechplus.io) |

BigTech+ repository အများစုသည် private ဖြစ်သည်။ Issue များကို ထို repository များအတွင်းမှ အလုပ်များကို ခြေရာခံရန်သာ အသုံးပြုပါ၊ public support channel အဖြစ် အသုံးမပြုပါနှင့်။

## Issue မဖွင့်မီ

1. ထပ်နေသော issue ရှိမရှိကို ရှိပြီးသား issue များတွင် ရှာဖွေပါ။
2. တောင်းဆိုလိုသည်နှင့် ကိုက်ညီသော issue form ကို အသုံးပြုပါ။
3. Bug များအတွက် ပြန်လည်ပြင်ဆင်ရမည့် အဆင့်များ (reproduction steps)၊ ဖြစ်သင့်သည့် behaviour နှင့် လက်ရှိ အမှန်တကယ် ဖြစ်ပေါ်နေသည့် behaviour(expected vs actual behavior)၊ environment အသေးစိတ်တို့ကို ထည့်သွင်းဖော်ပြပါ။
