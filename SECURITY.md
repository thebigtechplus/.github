# Security Policy

## Reporting

If you find a security issue in a BigTech+ system or repository, report it privately. Do not open a normal issue or discuss the details in a public channel.

**How to report:**

1. Email [security@bigtechplus.io](mailto:security@bigtechplus.io)
2. Or use GitHub private vulnerability reporting when it is enabled on the repository

Include the affected repository or system, version or commit if known, steps to reproduce, and impact.

We acknowledge reports within 48 hours and follow up with next steps.
